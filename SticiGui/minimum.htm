SticiGui requires an up-to-date standards-compliant browser. I recommend Mozilla Firefox 18.0 or higher. Current versions of Safari and Google Chrome work with most of the materials. The materials do not work reliably with Microsoft Internet Explorer.

In summary, you need:

  • Internet access, for example, either through an account specifically for this class, another university account, a commercial internet service provider (ISP), a public or university library, etc. If your connection to the network is slower than 28.8Kbps, some of the demonstrations and labs with large datasets can take quite a while to start up (several minutes), but once they are running, they should run smoothly.
  • An up-to-date web browser that supports HTML 4.01, frames, cookies, cascading style sheets, and JavaScript 1.8. SticiGui will not work correctly with older versions of browsers. Some problems will not be apparent except when you submit an assignment, so beware! If you are misgraded because you are using an out-of-date browser, your instructor might not be sympathetic.

    Not all features work reliably with Microsoft Internet Explorer; moreover, Internet Explorer is a security risk to you, so I recommend that you not use it for anything. You need to make sure that Java and JavaScript are "enabled" in your browser, not just that the software supports Java and JavaScript. How to enable Java, JavaScript, and cookies depends on which browser you use; see the documentation or help for your browser.

 

Not only must your browser support Java and JavaScript, but both must be enabled.
